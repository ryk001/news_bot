# Google新聞機器人 2024 版

想要每天早上在 Line 收到特定關鍵字的新聞嗎？
請跟著這個步驟做下去^^

## Step1. 申請 Line 金鑰
這樣才可以傳訊息給自己。
  - LINE Notify API 的金鑰，申請方式請參考: https://notify-bot.line.me/zh_TW

## Step2. 複製這個專案到自己的帳號
- 项目地址：[github/news_bot](https://github.com/ryk001/news_bot.git)
- 點擊右上角 Fork 專案至自己的帳號底下
